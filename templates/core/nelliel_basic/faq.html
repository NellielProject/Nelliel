FAQ

What is an imageboard?

Imageboards are a type of forum focused on posting and discussing images and other media. They are designed around the concept of anonymous posting as opposed to the registration required by most kinds of forums in order to participate.

What is Nelliel?

Nelliel is imageboard software intended to provide easy expandability and modern features without leaving behind the old school look and feel. It is PHP-based and originated as a modified version of Futallaby.

Tripcodes and Capcodes

Tripcodes are a unique hash for user identity on anonymous imageboards. Two primary types exist: traditional "insecure" tripcodes and secure tripcodes. The keys for each type can be used together but will only produce a single output.

Traditional tripcodes originated on 2channel in the early days of imageboards. It is widely supported and can allow using the same ID on multiple sites. However they use a weak hashing and can be easily compromised by modern PCs. Tripcodes are generated by putting # followed by any combination of characters in the name field.
Example: name#fgsfds

Secure tripcodes use a combination of stronger hashing and secret inputs that is much more secure. Unlike traditional tripcodes they are unique to each site and cannot be used elsewhere. Secure tripcodes are generated by putting ## followed by any combination of characters in the name field.
Example: name##fgsfds

Capcodes are a special identifier currently limited to staff. Capcodes do not generate a hash but produce an output based on board configuration. They are triggered by putting a single space followed by ## followed by another space then a valid combination of characters.
Example: name ## desu
